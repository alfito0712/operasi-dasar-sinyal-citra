## Deskripsi
Project ini merupakan implementasi operasi dasar pada sinyal 1D dan citra 2D menggunakan Python.

## Library

- NumPy
- Matplotlib
- OpenCV

## Cara Menjalankan

1. Install Python 3
2. Install library

```bash
pip install -r requirements.txt
```

3. Jalankan

```
notebook/operasi_sinyal_citra.ipynb
```

menggunakan Jupyter Notebook atau Google Colab.
